### **Nissan Epic Story DownLoad Program**  

---

- `config.py`: 접속 정보 및 JQL 설정
- `constants.py`: Jira 커스텀 필드 ID 매핑
- `jira_client.py`: (LOAD) Jira API 통신 및 데이터 수집
- `parser.py`: (PARSING) JSON 데이터를 우리가 원하는 딕셔너리 형태로 가공
- `excel_style.py`: (STYLE) 엑셀 시트의 너비, 색상, 틀 고정 등 디자인 적용
- `data_processor.py`: (SAVE) 데이터를 분류하여 엑셀 파일로 저장 실행
- `main.py`: 전체 프로세스 제어
